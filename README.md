# Poker Hands

A JavaScript application that uses the Deck of Cards API to deal and evaluate poker hands in the browser.

## Features
- Fetches a shuffled deck from the Deck of Cards API
- Deals a hand of 5 cards and displays card images in the browser
- Evaluates and displays the highest poker hand from the 5 cards
- Sort cards by suit or value
- Draw Again button to deal a new hand with fade animations
- Tracks remaining cards in the deck and disables draw when empty
- Wrapped in an IIFE to avoid polluting the global scope

## Technologies
- JavaScript
- HTML
- CSS
- Deck of Cards API

## API
- Deck of Cards: `https://deckofcardsapi.com/`

## How to Run
Open `index.html` in your browser — no installation required.
